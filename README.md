# 📊 Análise Férias funcionarios

## 🎯 Objetivo
Projeto desenvolvido para analisar quais funcionarios estão com férias em dia e férias vencidas.
Analisar o impacto financeiro que os funcionarios com férias vencidas pode gerar para a empresa.
Analisar quais setores tem mais funcionarios com ferias vencidas.

---

## 🛠 Ferramentas utilizadas
- Python
- Pandas
- Colab

---

## 📈 Análises realizadas
- Funcionrios com ferias em dia e ferias vencidas
- Impacto financeiro que cada funcionario com ferias a pagar gera a empresa
- Setor com maior custo de Ferias a pagar 
---

## 🔎 Insights
- dos 20 funionarios apenas 3 estão com as férias em dia
- o custo total de férias vencidas chega a R$353 mil 
- os funcionarios que mais geram custo são:
Felipe Carvalho
Camila Barbosa 
Patricia Alves
- O setor Facilites é o que mais gera custo em Ferias Vencidas 

---

## 🚀 Conclusão
A análise permitiu identificar um valor alto em ferias para ser paga, sendo necessario a realização de uma reunião para alinhar a programação de férias dos colaboradores, com o objetivo de otimizar custos e garantir que as operações da empresa não sofram impactos
